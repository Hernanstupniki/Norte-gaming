# Norte Gaming - Frontend Ecommerce

Frontend completo de ecommerce premium para Argentina construido con Next.js (App Router), TypeScript y Tailwind CSS.

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Estado local con React Context + localStorage
- Mock data realista (productos, marcas, FAQ, reseñas)

## Features implementadas

- Home premium con hero, marcas, categorías, destacados, ofertas, más vendidos, promo, beneficios y FAQ preview
- Catálogo con buscador, filtros (categoría, marca, precio) y ordenamiento
- Producto individual con galería mock, specs, reviews y relacionados
- Carrito lateral (drawer) + página de carrito
- Checkout visual por pasos
- Login y registro frontend (simulación)
- Favoritos
- Botón flotante de WhatsApp
- Responsive mobile first
